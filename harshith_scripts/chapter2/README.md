# Chapter 2: Tool Binding and Determinism

This directory contains the code assets and exercises for Chapter 2.

## Prerequisites & Setup
Make sure you have completed the global setup in the root `README.md`.
In addition to the previous chapter's setup, ensure you have `pydantic` installed:
```bash
pip install pydantic
```
And ensure your local Ollama is running (`ollama run llama3`).
