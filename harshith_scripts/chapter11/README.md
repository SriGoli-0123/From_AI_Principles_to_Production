# Chapter 11: Production Engineering: Cost, Guardrails, and Observability

This directory contains the code assets and exercises for Chapter 11.

## Prerequisites & Setup
All scripts in this chapter inherit the setup from previous chapters.
In addition, make sure you have:
```bash
pip install langgraph langchain-core langchain-openai
```
And make sure your local Ollama is running (`ollama run llama3`).
